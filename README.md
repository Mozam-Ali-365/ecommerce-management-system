# ecommerce-management-system
Python-based E-Commerce Management System with admin panel, cart system, checkout, stock management, and order history features.


## Features
- Admin Login
- Add Products
- Remove Products
- Update Stock
- Add to Cart
- Checkout System
- Order History
- JSON File Storage

- ## Technologies Used
- Python
- JSON
- Pandas

- ## How To Run
1. Install Python
2. Install pandas:
   pip install pandas
3. Run:
   python main.py
